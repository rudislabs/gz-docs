◆ kTestingProjectSourceDir

constexpr char kTestingProjectSourceDir[] = ""
constexpr

Constant referring to the project source dir of the current project.

For CMake builds, it is expected that this is injected via a buildsystem define of TESTING_PROJECT_SOURCE_DIR. This is done automatically for tfor tests using gz-cmake's gz_build_tests

For Bazel builds, it is expected to be empty
